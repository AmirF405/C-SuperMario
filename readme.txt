HyperMario Game

پروژه ماریو مابنی برنامه نویسی c
امیرحسین جوادی


نحوه بازی:

- کلید‌های حرکتی:
  - برای حرکت به سمت چپ از "A" استفاده کنید.
  - برای حرکت به سمت راست از "D" استفاده کنید.
  - برای پرش از "W" یا "Space" استفاده کنید.
  - برای حرکت به سمت پایین (نشستن) از "S" استفاده کنید.

- توضیحات اضافی:
  - کلیدهای راست و چپ را برای حرکت سریع نگه ندارید، بلکه هر بار فشار دهید.
  - زمانی که ماریو قارچ بخورد، اندازه او بزرگتر می‌شود.
  - پس از بزرگ شدن، با فشار دادن "S" یا دکمه پایین، ماریو به حالت نشسته درمی‌آید، اما در این حالت دیگر نمی‌تواند حرکت کند.



نحوه کامپایل و اجرای بازی:

برای کامپایل و اجرای بازی با استفاده از GCC، دستور زیر را اجرا کنید:

gcc game.c main.c users.c console_setting.c -o HyperMario.exe -lwinmm
